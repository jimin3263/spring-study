> 단축키 

`shift + f6`: 변수명 변경   
`command + option + V` : 변수 추출  
`command + option + M` : 메서드 추출  
`command + shift + T` : test파일 만듦  
`command + N` : getter, setter 등   
`ctrl + shift + enter`: 코드 자동완성  
`command + O` : 폴더 찾기  
`command + P`: 인자 값 보기  
`ctrl + T` : Refactor 관련   
`ctrl + R`: 이전 실행
